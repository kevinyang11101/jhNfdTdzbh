# 酒店管理系统 java1121

## 项目概述

酒店管理系统是一款基于Java和Vue的前后端分离的B/S架构软件，运用MVC模式设计，支持多种操作系统，适用于酒店行业的日常管理。

## 技术栈

### 系统环境
- Windows/Mac

### 开发环境
- IDEA
- JDK 1.8
- Maven
- Mysql

### 技术框架
- Java
- Mysql
- SpringBoot
- Mybatis
- Vue

## 功能模块

### 用户模块
- 用户登录
- 用户注册
- 预约住房
- 留言
- 查看公告
- 查看预约
- 入住查询

### 管理员模块
- 管理员管理
- 用户管理
- 房型管理
- 房间管理
- 预约管理
- 入住管理
- 留言管理
- 公告管理

## 系统角色
- 普通用户
- 管理员

## 运行环境
- 服务器：建议配置符合Java Web应用程序标准的服务器环境
- 客户端：现代Web浏览器

## 部署步骤
1. 准备好开发环境：IDEA、JDK 1.8、Maven、Mysql。
2. 克隆或下载项目源码。
3. 分别在前后端目录下执行构建命令。
4. 部署应用到服务器。
5. 根据文档进行环境配置和项目运行。

- 前后端分离技术，便于分工协作和扩展维护。
